# Reflexing

Build a sample reactive application in Rails.

## Installation

### Prerequisite

- ruby 2.7.2p137
- rails 6.1.0

### Install system dependencies

```sh
$> bundle install
$> yarn install
```

### Database creation

```sh
$> bundle exec rake db:create
$> bundle exec rake db:migrate
```

### Running project in local

```sh
$> rails s
```
